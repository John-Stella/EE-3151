# Control Systems Lab Repository

Welcome to the **Control Systems Lab Repository**. This repository contains lab manuals, experimental code, and supporting resources for hands-on learning in control systems.

Each lab is organized into its own directory, with a dedicated `README.md` that serves as the **lab manual**. Within each lab, you’ll find structured parts containing code and instructions for running experiments.
